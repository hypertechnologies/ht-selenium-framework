implicitWaitTimeout = Time in milliseconds for WebDriver to implicitly wait
explicitWaitTimeout = Time in milliseconds for WebDriver to explicitly wait
ignoreCertificateError = If true, WebDriver will ignore certificate error. This should be false in most cases.
browsers = comma separated list of browser name to execute the test suite.
screenSize = size of the browser window when running tests.
Suite = Objects of suite containing relative directory and whether you want to skip or not.
excelIndexes = list of indexes for different columns in excel sheet.
